# App Concept

## Name
Space Shooter

## Concept
ブラウザで即遊べる、スペースインベーダー風シューティングゲーム

## Description
プレイヤーが宇宙船を操作し、画面上部から降りてくる敵編隊を撃破する。
シンプルな操作で誰でも楽しめ、スコアを競うカジュアルゲーム。
インストール不要でブラウザ上で動作する。

## Target Users
ブラウザでサクッと遊びたいカジュアルゲーマー全般

## Use Cases
- 隙間時間にブラウザで遊ぶ
- スコアを更新するためにリプレイする
- キーボード操作でシンプルに楽しむ

## Design Philosophy
シンプル・高速・レトロ感のある見た目。余計なUIはなく、ゲームに集中できる構成。

## Constraints
MVPでやらないこと：
- オンラインランキング
- サウンド・BGM
- モバイルタッチ操作
- セーブデータの永続化

## Input
- キーボード（←→移動、スペース長押しで連射）
- ゲームパッド（左スティック or 十字キーで移動、Aボタン長押しで連射）
